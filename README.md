# Hello World Sample App

Nothing much, except starting a web server at certain port.

## How to run

#### Windows

```bash
set PORT=8083
set INSTANCE_ID=some-text
go run main.go
```

#### Nix/Linux

```bash
export PORT=8083
export INSTANCE_ID=some-text
go run main.go
```

Or,

```bash
PORT=8083 INSTANCE_ID=some-text go run main.go
```
